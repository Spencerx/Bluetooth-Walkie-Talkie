README FILE

An executable (.apk) file is uploaded in this directory to install the Bluetooth Walkie Talkie app in your Android Mobile Phone.

WalkieTalkie.apk

This executable apk file installs Bluetooth Walkie Talkie app on your android phone.

Instructions:

1. Make sure that you have the proper permissions to run apps from other sources. In your phone, open Settings->Security->"Unknown Sources" then enable 'Allow installation of apps from sources other than the play store'.

2. Browse to this directory in your android phone's browser and click on the WalkieTalkie.apk file to download and install it.
